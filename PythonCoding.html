The task was simple. I was given 3 classes. "Enemy", "Monster" and "Alien" with the properties name and lives. I was also given a function called hit. Each hit would take 1 life.

So I inherited the classes monster and alien from the Enemy class and used a while loop to call the hit function either on the Monster or the Alien.

My mistake was to put paranthesis after the m and a (wich were given variables for the funtions Monster() and Alien(). That resulted in putting out the memory location of the function called in the while loop, even though the function got called the right amount of times. By deleting the paranthesis, the function would execute by calling the hit function with it's corresponding values and give me a successful test output for the problem.
